# NYC Trip Record Data Project

## Overview
This project is a Dataform project that aggregates nyc Taxi data on BigQuery
and builds a data for BI tools.

## Architecture
-warehouse: BigQuery
-orchestration: Cloud Scheduler + Dataform

## Directory Structure
-defenitions/
-staging/
-marts/
-includes/
-assertions/

## Execution Environment
-Production: main
-Development: feature/*

## Naming Conventions
-stg_: Staging
-mart_: Data Mart
